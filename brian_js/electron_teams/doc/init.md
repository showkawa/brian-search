## base framework 
https://github.com/electron-vite/electron-vite-react

## tech stack
 - vite, react, electron
 - shadcn UI
 - tailwindcss for customize styles
 - SWR for sync data fetching
 - ky for fetch API
 - vite-plugin-pages for file-based routing
 - zustand for state management
